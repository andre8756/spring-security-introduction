# Spring Security Introduction

Este repositório contém uma introdução prática ao Spring Security, com foco nos conceitos básicos de segurança em aplicações Spring Boot, como autenticação e autorização.

O projeto foi desenvolvido com fins educacionais, servindo como base para estudos iniciais sobre Spring Security.

## Conteúdo

- Exemplo simples utilizando Spring Boot e Spring Security  
- Configuração básica de segurança  
- Proteção de rotas  
- Estrutura introdutória e didática  

## Base de Estudo

Este projeto foi criado com base no artigo:

**Spring Boot Security Step by Step**  
Artigo disponível no Medium.

## Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/andre8756/spring-security-introduction.git
   ```

2. Importe o projeto em sua IDE de preferência (IntelliJ, Eclipse ou VS Code)

3. Execute a aplicação como um projeto Spring Boot

4. Acesse a aplicação e observe o funcionamento do Spring Security

## Tecnologias Utilizadas

- Java  
- Spring Boot  
- Spring Security  

## Objetivo

Fornecer uma base simples e funcional para compreender os fundamentos do Spring Security e servir como ponto de partida para implementações mais avançadas.
